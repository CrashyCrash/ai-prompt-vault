**Curated collection of powerful AI prompts for developers, writers, marketers, and creators.**

## 🎯 Features

- **20+ curated prompts** across 6 categories
- **Search & filter** by title, content, or tags
- **Copy to clipboard** with one click
- **Favorites** saved to localStorage
- **Responsive** — works on desktop and mobile
- **Dark theme** with glassmorphism design

## 📂 Categories

| Category | Prompts |
|----------|---------|
| 💻 Coding | Code review, React components, API design, debugging, testing |
| ✍️ Writing | Blog posts, README generator, email drafting |
| 📣 Marketing | Landing page copy, social media calendar, Product Hunt launch |
| 🔬 Research | Competitor analysis, technology deep dives, trend analysis |
| 🎨 Image Gen | Product mockups, logo design, social media banners |
| ⚡ Productivity | Project planning, meeting notes, daily standups |

## 🚀 Tech Stack

- **React 19** + **TypeScript 5.9**
- **Vite 8** — Lightning-fast builds
- **Tailwind CSS 4** — Dark theme with custom design tokens
- **Lucide React** — Beautiful icons

## 📦 Development

```bash
pnpm install
```
